# 趋势金字塔EA - 5分钟快速上手

## 📦 1. 安装（1分钟）

### 复制文件到MT4

```
1. 找到 MT4 数据文件夹
   - MT4菜单 → 文件 → 打开数据文件夹

2. 复制策略文件：
   src/strategies/PyramidTrend_EA.mq4 
   → MQL4/Experts/

3. 复制依赖文件：
   src/strategies/pyramid_trend.mqh 
   → MQL4/Include/strategies/
   
   src/risk/trade_filters.mqh 
   → MQL4/Include/risk/

4. 重启MT4（或点击导航器刷新）
```

## 🚀 3. 启动EA（1分钟）

1. **打开图表**
   - 推荐品种：EURUSD, GBPUSD, XAUUSD
   - 推荐周期：H1, H4

2. **加载EA**
   - 从导航器拖动 `PyramidTrend_EA` 到图表

3. **设置权限**
   - ✅ 勾选"允许实时自动交易"
   - ✅ 勾选"允许导入DLL"（如果提示）
   - 点击"确定"

4. **确认运行**

## ⚠️ 常见问题速查

### Q: 图表右上角显示哭脸 ☹️
**A**: 未启用自动交易
- 工具 → 选项 → EA交易 → 勾选"允许自动交易"

### Q: 一直不开仓
**A**: 等待趋势确认
- 策略需要明确趋势（ADX>25）
- 查看日志是否有拒绝原因
- 盘整市场不会开仓（这是正常的）

### Q: 止损太大/太小
**A**: 基于ATR动态计算
- 调整 `Trail Stop ATR Multiplier` 参数
- 默认2.5倍，可改为2.0（更紧）或3.0（更松）

### Q: 如何停止EA
**A**: 三种方式
1. 删除图表上的EA图标
2. 取消勾选"允许自动交易"
3. 修改参数触发重新加载

- **稳健设置**：最大回撤 5%-10%
- **激进设置**：最大回撤 10%-15%
